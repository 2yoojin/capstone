# 프로젝트 개요
**주제 설정** :
한강 수위 이상치 예측을 통한 홍수 및 가뭄 감지

**수집 방법** : 
한강홍수통제소 제공 데이터 활용

**측정 기간** : 
2010년~

**측정 장소** : 
한강 내 대교별 측정

# 프로젝트 목적

1. **수준별 대비 가능**:  여름에 하천홍수, 집중호우등 여러가지 상황으로 인한 침수 발생가능 
이상치 탐지가 아닌, 수위 데이터의 이상치값을 정확하게 예측함으로써 하천 범람으로 인한 수준별 대비가능

2. **여러 재난 상황 대응** : 가뭄으로 인한 수위 하락도 하천의 수위 데이터의 이상치값으로 판단 가능할 것으로 예상
